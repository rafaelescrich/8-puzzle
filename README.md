# 8-puzzle

Implementação em Go para encontrar o custo da solução usando A*.

## Uso

### Linux e OS X

```bash
go build -o 8-puzzle-linux-x64
./8-puzzle-linux-x64
```

Cross compile from linux to OS X

```bash
GOOS=darwin GOARCH=amd64 go build -o 8-puzzle-mac
./8-puzzle-mac
```
